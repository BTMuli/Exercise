【问题描述】

两个整数之间的汉明距离指的是这两个数字对应二进制位不同的位置的数目。

【输入形式】

给出两个整数 x 和 y(0 ≤ x, y < 231)，用空格分隔

【输出形式】

输出它们之间的汉明距离
【样例输入】

```
1 4
```

【样例输出】

```
2
```

【样例说明】

1的二进制表示为（00000000 00000000 00000000 00000001）

4的二进制表示为（00000000 00000000 00000000 00000100）

他们的第一位和第三位不一致，所以它们的汉明距离为2

【解题思路】

+ 计算出每一位的xi,yi，若两者不等，ans++

【源代码】（通过样例测试）：

```c++
#include<iostream>
using namespace std;
int main()
{
    int x, y, xi, yi, ans;
    ans = 0;
    cin >> x >> y;
    for (int i = 0;i<32; i++)
    {
        xi = x % 2;
        yi = y % 2;
        if(xi!= yi)
            ans++;
        x = (x - xi) / 2;
        y = (y - yi) / 2;
    }
    cout << ans;
    return 0;
}
```
